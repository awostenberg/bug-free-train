## getting started

### first time
- python -m venv .venv
- source .venv/bin/activate   #to get pip
- pip install -r requirements.txt

### thereafter
- source .venv/bin/activate


## to run 
- ptw . # to run all
- ptw -clear . -- test_gen.py # to run just the one
